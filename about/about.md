this is an app to help you build software demos. 

should support several scenarios.

1. the user provides
    - a script
    - key to audio generator
    - a url to an app

2. The app takes the script, allows user to update the script for pronounciation. the initial script is used later for output.

3. app takes completed script and the app makes the audio script based on user prefered voice selection. 

4. app produces points an clicks and generates video. user provides notes on where to zoom in or what to highlight.

4. audio used for srt, audio translated into multiple languages
