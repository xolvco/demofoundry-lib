# DemoFoundry

DemoFoundry helps you build **software demo videos** from a written script — it narrates,
drives your app automatically, and produces a finished video with subtitles in multiple
languages.

## What it does

1. You provide a **narration script**, a **TTS (text-to-speech) API key**, and the **URL** of the app to demo.
2. Edit the script to fix **pronunciation** — the original text is kept and reused later as captions.
3. Pick a **voice** and generate the narration audio.
4. Mark the **clicks, navigation, zoom, and highlight** points for the walkthrough; DemoFoundry
   drives your app and records the video.
5. Export the finished **video**, plus **SRT subtitles** translated into multiple languages.
