# Project memory

Durable context for DemoFoundry — the **why** behind decisions and the running list of open items,
kept in the repo so it survives across sessions and contributors.

- [decisions.md](decisions.md) — decision log (what we chose and why). ADR-lite.
- [notes.md](notes.md) — running comments: open questions, next steps, things to watch.

This complements (does not duplicate) the reference docs under [`../docs/`](../docs/): the docs
describe *how the system works today*; this folder records *why it's that way* and *what's next*.
